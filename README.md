**Goat Wordle**

A goat-themed Wordle clone for iOS, built in SwiftUI. Guess the five-letter word in six tries — now with significantly more goats.

**Features**
- Classic Wordle rules: six guesses, color-coded feedback for correct, misplaced, and wrong letters
- Custom on-screen keyboard that tracks which letters you've used
- Tile flip animations and celebratory toast messages
- Goat photos. Several of them.


**Built with**
- Swift / SwiftUI (MVVM architecture)
- Custom animations with SwiftUI transitions


**Why**

A for-fun project to practice SwiftUI state management, custom animations, and building game UI from scratch — no game engine, just views.
